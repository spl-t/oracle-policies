# Oracle

All-in-one Persona 5 Royal companion bot for Discord.

## Features

- **Personas** — Stats, skills, weaknesses, and fusion info from the full compendium
- **Skills & Traits** — Descriptions and which personas learn them
- **Confidants** — Conversation choices and rank-up guides for all characters
- **Exams** — All classroom and exam answers by month
- **Crosswords** — Solutions for every crossword puzzle
- **Jazz Jin** — Daily drink schedules, Sunday skill schedules, and live singer dates

## Commands

| Command | Description |
|---------|-------------|
| `/persona` | Look up any persona |
| `/skill` | Search skills and traits |
| `/confidant` | View confidant guides and conversations |
| `/exam` | Get exam answers |
| `/crossword` | Crossword solutions |
| `/jazz` | Jazz Jin schedules |
| `/help` | Full command list |
